# Radio

## Building

### Requirements

- MinGW >= 13.1.0 64-bit
- CMake >= 3.22
- Qt >= 6.9
- mpv >= 0.41.0
    - `libmpv` directory with the `libmpv-2.dll` file and the `include/` directory
    - [Get from here](https://mpv.io/installation), the needed archive file is usually marked as `mpv-dev-x86_64`
